## Portfolio

Ce projet est un portfolio personnel développé avec **Laravel**, conçu pour présenter mes compétences, mes projets et mon parcours professionnel.  
Il met en avant une architecture propre, un rendu dynamique des contenus et une stack de développement moderne basée sur Docker pour un environnement cohérent et maîtrisé.

Ce portfolio sert à la fois de vitrine professionnelle et de démonstration technique de mon savoir-faire en développement web.
